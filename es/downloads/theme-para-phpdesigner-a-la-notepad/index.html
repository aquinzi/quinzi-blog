Theme para phpDesigner a la notepad++

Publicado
2011-06-05
Source
Obtener código

Probando nuevas IDEs para el desarrollo PHP me topé con phpDesigner es muy lindo, pero no tiene code folding (!).

Otra cosa fea es el theme que viene por defecto para el syntax-highlater, y, para decir la verdad, me acostumbré al de notepad++. En mi búsqueda no encontré ninguno parecido y menos encontrar uno "light" cuando el 99% son "dark" themes, asique me decidi a hacer el mio. Traté de copiar lo mas que pude los colores.

Acá imágenes re bonitas de muestra:


Muestra CSS


Muestra HTML


Muestra JavaScriptL


Muestra PHP

Les gustó? si, no? bueno, se puede descargar aca. Tiene para CSS, HTML, PHP y JS.

¿Cómo instalarlo?

  • Descomprimen el .zip
  • Abren el phpDesigner
  • Aprietan ctrl+e o van a Tools->preferences
  • Van a la parte de Syntax Highlighters. Ahi seleccionan CSS y van a importar. Seleccionan el archivo syntax_css2.dat
  • Teoriamente con uno solo alcanza para todos los lenguajes, pero si no es asi, hagan el mismo procedimiento para php, html y javascript con los archivos correspondientes.
  • Ahora siguen los pasos que estan en el txt para completar la customización
