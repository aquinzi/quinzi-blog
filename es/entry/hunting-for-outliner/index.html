Hunting for an outliner/task list manager

Publicado
2012-11-05
Source
Obtener código

This was suppose to be a review of the program below but this idea is better as it joins two posts in one (the world is in economic crisis)

I think if you follow me on the twitters you must be super tired (and probably muted) my tweets about Bonsai and a replacement for it (taking the consideration that is abandoware now).

I have tried many during the last year? (don't remember) many programs (apps seem to be called now), servicies; none of them I liked.

Lets make a list of my requirements.

Cute list of author's requirements:

  • in place adding and editing
  • filtering
  • notes for items
  • subtasks / hierarchical list
  • display options/workspaces/desktops
  • outliner + to do list (same program) with project/tasks item (progress bar)
  • priority
  • dates (created, due, completed)
  • keywords / category / tags
  • export
  • no database format if possible, but I take sqlite.
  • import
  • Easy keyboard shortcuts or let me change them
  • sorting
  • no web version, unless is very uncluttered/simple/minimalistic (like workflowy) or give me a PC version
  • free
  • search
  • print
  • fast loading/opening

Nice to have but not necessary:

  • zoom into branch
  • link to files
  • customize colors by priority, keyword, dates...
  • portable
  • android version with easy sync (free)
  • Insert date/time

That's it... I don't think I ask for too much. There is this program that it's very close to that but it has it downsides:

Program in question: My Life Organized (MLO for short).

It has:

  • In place editing (must click exactly on the word to make it "in place", otherwise, double clicking)
  • filtering
  • notes for items
  • subtasks / hierarchical list
  • display options/workspaces/desktops
  • outliner + to do list (same program) with project/tasks item (progress bar) (kind of, the project option is paid)
  • priority (but as it's GTD oriented, it has the whole important/urgency thing)
  • dates (created, due, completed)
  • keywords / category / tags (but, again, as GTD oriented, has them as contexts)
  • export
  • import (depends of version)
  • PC version
  • free (most of the features are free, there are other versions)
  • search
  • zoom into branch (not in Light/Free version)
  • portable
  • print
  • fast loading/opening
  • Insert date/time

It misses:

  • In place adding (most important)
  • no database/sqlite. (You can export to xml but you lose some funcionality)
  • Easy keyboard shortcuts or let me change them (alt+shift+right to indent? really?)
  • sorting (kind of basic)
  • link to files
  • customize colors by priority, keyword, dates...
  • android version with easy sync (free). (It does have an android version, but the syncing is not free)

But the paid version has an Rapid Task Entry Dialog, which is better than press insert->task entry -> enter -> insert ->task entry -> enter -> alt+shift+right. You can add sub items to folders in that dialog (I couldn't make it work with a normal task). This makes up for the fast and easy add task item, but again, is a paid feature.

Here is a screenshot on where Bonsai shone:

That's an outline/task list for this college project crappy program (don't make me remember that, please).

See why is important the easy entry? I wouldn't make that with the "please, press this key to make a new item. Now enter some text. Thanks, now, if you want to make it a subtask, please, press this long keyboard combination". I - just - wouldn't. I prefer making the list with a "rapid entry" option (ex. call Timmy at 10 pm tomorrow) in text format and then importing, before going with that hell of keyboards combination.

Also like the distintion of the different types of items. You have the task/project (progress bar) and the To Do item. The one you can't see is the outline that it doesn't show any icon.

The notes can be shown all the time to the side or bottom or have them as an icon, where you click/hover and it shows.

Filters, easy dates change/edit... all in plain sight.

The keyboards combinations where very easy: tab to indent, shift+tab to unindent, enter makes new item, alt+enter enters to the item's properties. +/- to expand/collapse. shift+-/+ to collapse/expand all, change priorities, change type of item... all very easy to remember.

It was a pleassure to use it. But oh well, I think it's time to move on. I will have it installed just in case, maybe I need to use it for another project management.

So, what will you do, dear author?

Well... no idea. I might start using MLO for hierarchical to do lists. I don't think it worth it as an outline, unless I use the Rapid Task Entry Dialog.

As for outlines, I was testing UV outliner. It's nice, simple, fast. It also works for little and simple to do lists. The other one, a web service, Workflowy. It's the only web app that I see myself using, it loads fast, minimalistic view, so plain. Works with my connection (take that Google!).

Or I just forget everything and use a wiki instead.

I love wikis...

Really much... very flexible.
