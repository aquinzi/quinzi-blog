Templates en Zim Desktop

Publicado
2011-07-15
Source
Obtener código

Zim Desktop es un programa de wiki de escritorio multiplataforma. En los foros decian que iban a presentar templates para las paginas, segun namespaces (como cualquier sistema de wiki). Desconozco si las ultimas versiones ya lo traen, pero en la version para Windows no está. Acá traigo una solución.

Nota: es un template para toooodas las paginas nuevas, no por namespace.

Buscar donde se instaló el programa, e ir a: data\templates\wiki. Ahi hay un archivo de texto con el nombre Default.txt. Ese es el archivo a modificar para que cada vez que pongas "Crear una nueva pagina" salga con ese template.

En esta parte de la página oficial tenes un poco mas sobre el tema de los templates como variables y sintaxis.
