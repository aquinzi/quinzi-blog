Wordpress XML a HTML

Posted
2010-09-18
Source
Get source

Esto es simple: en su dashboard van a export/exportar. Ponen todos los filtros que quieran, y exportan.

Qué sale? un lindo xml de unos megas. Cómo lo leo/paso a HTML? con el wpxslgui.

El wpxslgui es un programita bien simple. Lo unico que tienen que hacer es, en la primer cajita, seleccionar donde esta el xml bonito que generó Wordpress. Donde dice transformation elijen el tipo de transformacion. En output file ponen donde y cómo quieren que se llame. Una vez ingresado los datos correctos (?) le dan a Generate. El programa no te avisa/dice el porcentaje del proceso asi que hagan calculos ustedes (?). A mi me tardó menos de 30 segundos, un xml de 12MB mas o menos.
