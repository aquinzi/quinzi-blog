Problemas de actualización automática de Wordpress

Posted
2009-12-06
Source
Get source

Quién quiere bajar un plugin, descomprimirlo, abrir el FTP, acceder, copiar el plugin ahi, poner reemplazar.. no a nadie, a mi tampoco.

Estando en este host (Byethost) no podía actualizar automáticamente Wordpress, tampoco los plugins. Recién ahora (hace unos minutitos nu más) encontré la solución: en el archivo wp-config.php, antes de terminar (?>) se pone esto:

if(is_admin()) {
add_filter('filesystem_method', create_function('$a', 'return
"direct";' ));

define( 'FS_CHMOD_DIR', 0751 );
}

Para más información, saber porque pasa, etc... Aca fue donde lo saqué
