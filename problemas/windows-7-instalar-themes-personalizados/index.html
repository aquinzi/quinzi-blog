Windows 7: Instalar Temas/Themes personalizados

Posted
2010-02-13
Source
Get source

Aburridos, asqueados del tema/theme/estilo visual de Windows 7 Home basic? ese azulón/baby blue? yo si, me harté de verlo y comparto cómo instalar los custom/personalizados temas/vs/themes.

Antes que nada, el Sistema Operativo que tengo es Windows 7 Home Basic 64 bits (build 7600) y funciona, asique adelante, no tengan miedo!.

Instrucciones

  1. Desactivar el UAC (Configuración de Control del Usuario... en realidad es Cambiar Configuración de Control del Usuario. Si tienen problemas para encontrarlo, solo tipeen UAC en la cajita de búsqueda del menú inicio). Lo que tienen que hacer es desactivarlo o ponerlo en la segunda barrita (contando desde abajo).
  2. Instalar el UxStyle Core, (acordarse de instalar el que es de acuerdo a su estructura). Leer los Términos de Servicio y si se está de acuerdo, siguiente. Si no se está de acuerdo, igual.
  3. Poner la configuración de UAC como se tenia anteriormente (opcional)
  4. La magia ocurrió, hora de descargarse los temas para cambiar el horrible que viene por defecto. Estos se descomprimen en X:\Windows\Resources\Themes (siendo X donde se tiene instalado el Sistema Operativo)
  5. Ir al Panel de Control>Cambiar Combinación de colores y elegir el que más te guste. Acá hay un problemita, capaz que soy yo pero, los themes tienen el mismo nombre (Windows 7 Standard/Basic) asi que no queda otra que ir clickeando y ver como queda en la muestra :)
  6. Disfrutar!

Un poco de blababla

Existen parches, pero todos saben que es un 50% de que anden/no pase nada y el resto que se joda arruine algo y tener que, a veces, re instalar el Sistema Operativo. Este programita es un servicio (lo pueden encontrar en el Administrador de tareas>servicios, se llama UnsignedThemes.exe) que ocupa un poquito de memoria y "es un poco más seguro" (para el sistema) que los parches. El proceso "engaña" a Windows en pensar que todos los temas estan firmados y asi poder usar los personalizados.
