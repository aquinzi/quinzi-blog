AutoHotkey + Nueva Carpeta

Posted
2009-05-09
Source
Get source

Uso Windows. Nada malo con eso (por ahora :P ) . Qué se extraña de otros Sistemas Operativos? Keyboard Shortcuts / Comamands (Atajos de teclado / Comandos, Ctrl + C…) para una nueva carpeta. Sé que se puede apretando: AppMenu (ese que funciona como el botón derecho…), arriba x 2 (manera mas corta), enter/derecha, enter. O desde el menu de arriba en una carpeta: alt, enter/abajo, enter/derecha, abajo. Pero es mucho lio, y las computadoras se crearon para automatizar el trabajo (?), asique pienso yo que con dos teclitas se soluciona parte del problema.

El programa a descargar es el Autohotkey (que uso yo. Es gratuito, OpenSource y para Windows). Dos versiones: exe y zip. Si queres crear scripts rápidos, optá por el exe. Si no te interesa/no te molesta la rapidez (solo son un par de pasitos mas, tampoco es mucho) anda con el Zip (soy pro-comprimido).

Una vez que lo instalaste, en una carpetita/directorito donde vas a guardar los scripts hacé: boton derecho>crear nuevo AutoHotkey Script. Si bajaste el Zip, creas un archivo de texto con la extensión .ahk.

Te aparece re bonito el archivito, le cambias el nombre y todo eso (yo le puse “nuevacarpeta” porque soy muuy original y tengo muuuucha creatividad). Sobre éste, botón derecho>Edit Script. Te aparece el bloc de notas (va, yo lo tengo así… cualqui, en vez de “Edit Script” vas a “Abrir con Notepad/Bloc de notas”). El Archivito contiene lo siguiente:

Ahí podes (y creo que no afecta, ya que es para distribuir solamente) completar para que quede mas Pro y solamente empezar a escribir abajo de todo, o eliminarlo completamente si solamente lo vas a usar vos.

Lo que hay que hacer ahora es utilizar los macros, trabajar todo con teclado y traducirlo al código propio. Como soy una buena persona copio y pego el código:

n::send {AppsKey}{up}{up}{right}

Explicación:

#n es el Shortcut. En este caso es: Win+n. Obviamente se puede cambiar al gusto de cada persona: \^ = ctrl / control; ! = alt.

sendes para enviar “la info”, solamente ponelo, sino no funciona.

{AppsKey}{up}{up}{right}{enter} = {botón derecho sobre una parte limpia de la ventana/escritorio} {vamos para arriba} {vamos para arriba} {vamos para la derecha/hacemos clic} {enter/hacemos click}. Es la traduccion de lo que hariamos por mouse, o por la descripción por teclado que habia escrito al principio.

Una vez esto, hacemos (lo que hago) botón derecho sobre el archvito, vamos a: abrir con (aunque también se puede modificar para que, al hacer doble click sobre él abra el programa), AutoHotkey.

Se nos carga el programita y ya está. Apretas (previamente deseleccionado todo) Win+N y tenes tu carpetita nueva, aguardando renombrarla.

Podes poner que el AutoHotkey se cargue con Windows y asi automaticamente el script, compilarlo y que te quede un exe que lo pones en la carpeta de Inicio (y asi no necesitar el prog. para correrlo), o bajar el que está en el archivo que pongo mas abajo que contiene: el ahk y el exe con un txt por si no sabes donde hay que ponerlo.

Obviamente la potencia del App no se queda solo en eso, también se puede susplantar texto mientras se escribe o poner una palabra, apretar un trigger y que se susplante con un texto. Esto seguro lo explicaré mas adelante.
