This is, some how, to mark my japanese learning progress. This way, I "motivate" to learn a bit more. Everything started when I stopped learning for like 6-8 months.

This poster used to show the chapters done and to go for finishing the book showed there ([that I've finished][]). I thought about doing the same for RTK3 but as I will not add every kanji there, it will be a waste of time.

This other indicate the lessons that I know from the book shown in the right side. It's been a long time since I take a look to that one. Anyways, lately, I've centered on kanji so I could finish the book (RTK) and haven't read any grammar.

These 3 pages are the kanji from RTK. Was the first "poster" I made. This one is going to die soon for one that's prettier.

This "poster" contains 3 frequency kanji list. In total, there are about 18xx kanji (because I didn't want to print a 3rd one XD).

This mini progress bar would represent mi level. I took it from a webpage where the guy did this RPG learning style thing. Now that I see, Beginner should be in a 60%-80%. The star represents where my engish level would be, according to this RPG guide.

This is not a poster, even though I've tried to do something similar last year... unsuccessfully.With this, you mark the days when you've done somehing related to yourr learning. This way, when you feel like doing nothing, you see the little squares and say: "mmm nop! Don't wanna have blank squares!" and you do something :P

This goes with the above thing. It's like a journal. Tecnically, you write down what and how much you've done, how you felt, and/or other stuff like study time. As I don't want to be very detailed, so I can write everything down quickly, I just specificate what I've done.

And that's it!. Maybe this helps someone and give an idea for... something. 頑張って！
