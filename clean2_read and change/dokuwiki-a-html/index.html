Pasar de DokuWiki syntax a html

Posted
2011-06-11
Source
Get source

Amás la sintaxis de las wiki. Te acostumbraste a una y queres parsear la sintaxis de esa wiki que tenes en un txt a html? Si usas DokuWiki te salvé :P

La cosa es asi: pones el php donde te dicen, usas la cmd, cli.... para pasarle el txt et voilà! html parseado =B It's magic!

Requisitos:

Tener DokuWiki instalado, andando y amando(lo) (?)

  • Este script:
<?php
if ('cli' != php_sapi_name()) die();
ini_set('memory_limit','128M');
if(!defined('DOKU_INC')) define('DOKU_INC',realpath(dirname(__FILE__).'/../').'/');
define('NOSESSION',1);
require_once(DOKU_INC.'inc/init.php');
require_once(DOKU_INC.'inc/common.php');
require_once(DOKU_INC.'inc/parserutils.php');
$source = stream_get_contents(STDIN);
$info = array();echo p_render('xhtml',p_get_instructions($source),$info);
?>
  • Guardarlo dentro de la carpeta de DokuWiki\bin.
  • Tener el php en el PATH del SO para ejecutarlo desde el cmd/cli

Uso

php path/a_DokuWiki/bin/dokucli.php < input.txt > output.html

Happy parsing!
